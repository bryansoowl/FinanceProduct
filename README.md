# FinanceProduct Analysis – Power BI Practice

## Overview

This project is a simple **Power BI practice project** where I cleaned and prepared a financial dataset, created new calculated fields, and built a basic interactive report to practice data visualization and data transformation skills.

## Dataset

The dataset used in this project was obtained from Kaggle:

https://www.kaggle.com/discussions/general/402148

It contains sample financial data including:

* Segment
* Country
* Product
* Sales
* Profit
* Units Sold
* Date and time-related fields

## Data Cleaning & Transformation

Before building the report, the dataset was cleaned and transformed using **Power Query in Power BI**.

The following steps were performed:

* Corrected the **Date column** using the `Year` and `Month Number` fields.
* Created and replaced **Corrected Date** column to ensure the date values reflect the correct month.
* Created a **YearQuarte**
